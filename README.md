# WasteWise

## Setup Instructions

### Firebase Authentication (Optional)

For local hackathon demo, Firebase is optional. 
To run without Firebase, leave `VITE_FIREBASE_*` variables empty in `frontend/.env` and use **"Continue as Demo User"** to log in instantly.

To enable Google login:
1. Add valid Firebase web app configuration to `frontend/.env`.
2. Authorize `localhost` in your Firebase Console authentication settings.
